# Actividad 2 — Transformada de Fourier

Este repositorio contiene el código en Python para la simulación y análisis de señales usando la Transformada de Fourier.  

## Contenido
- Señal senoidal en el dominio del tiempo
- FFT: magnitud y fase
- Propiedad de linealidad
- Propiedad de desplazamiento en el tiempo
- Propiedad de escalamiento en frecuencia

## Herramientas
- Python 3
- NumPy
- Matplotlib
- Google Colab
